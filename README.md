# TIC-TAC-TOE# 🎮 Tic Tac Toe

A modern and interactive **Tic Tac Toe** game built with **HTML**, **CSS**, and **JavaScript**. Play against a friend in **2-Player mode** or challenge the **AI** in **Player vs AI** mode.

## ✨ Features

- 🎲 Two Game Modes
  - 👥 Player vs Player
  - 🤖 Player vs AI
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Restart/New Game functionality
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🎨 Clean and modern user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/tic-tac-toe.git
```

2. Open the project folder.

3. Launch `index.html` in your preferred web browser.

No installation or external libraries are required.

## 🎮 How to Play

### 👥 Player vs Player
- Player **X** starts the game.
- Players take turns placing their marks.
- The first player to align **three symbols** horizontally, vertically, or diagonally wins.

### 🤖 Player vs AI
- Play as **X** against the computer (**O**).
- After your move, the AI automatically makes its move.
- Try to outsmart the AI and win!
